About Us

In this project our team was able to apply our HTML skills by developing a static site generated by Eleventy.
Our site showcases a collection of photos that have been extraced from a .json file provided by Pexels.
Each photo has its own page that contains a larger, captioned version of the photo along with more information about the photographer including their Pexels page and ID.
